# databases

Предметной областью базы данных являются характеристики графических редакторов.
Для хранения информации о каждом графическом редакторе предложена структура, содержащая поля – название, тип дизайна, разработчик, поддерживаемые форматы, цена, веб-сайт, версия API и системные требования.

# Задачи:
- Обосновать выбор структуры данных для хранения информации о графических редакторах, включающей 8 различных полей. 
- Реализовать файловый ввод-вывод в читаемом текстовом формате и 
- Отладить и протестировать программу.

  ##  Возможности
- ✅ Добавление/удаление записей о редакторах
- ✅ Поиск по типу дизайна (2D/3D/UI/UX)
- ✅ Поиск по цене + формату файла
- ✅ Сортировка по цене (qsort)
- ✅ Сохранение/загрузка в текстовые файлы
- ✅ Генерация тестовых данных
